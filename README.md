                                               🏦 Bank Management System (Java OOP Project)

A lightweight Bank Management System built using core Java with a strong focus on Object-Oriented Programming (OOP) principles and HTTP server implementation.
<br>
This project demonstrates how backend systems can be designed from scratch using Java without relying on heavy frameworks.
<br><br>
    🚀 Live Application
<br>

<br><br>
The objective of this project is to build a simple and functional Bank Management System using core Java concepts.<br>
It focuses on:<br>
.) Applying Object-Oriented Programming (OOP) to model real-world banking operations<br>
.) Understanding how a backend system processes user actions<br>
.) Connecting a basic user interface with backend logic<br>
.) Strength<br>
<br>
                                                        Core Java & OOP Concepts Used<br>
                                                        Encapsulation<br>
                                                        Data Abstraction<br>
                                                        Object-Oriented Design<br>
<br><br>
Flow in the project--<br>
User (browser)<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;↓<br>
index.html (frontend)<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;↓<br>
Request sent (Create / Deposit / Withdraw)<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;↓<br>
Server.java (backend + server)<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;↓<br>
Processes using Account objects (OOP)<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;↓<br>
Response sent back<br>
